# Philly Spot — Premium Restaurant Website

A professional, modern website for Philly Spot cheesesteak restaurant in Hillsboro, Oregon.

## ✨ Features

- **Sticky Navigation** — Clean navbar with quick access to menu, location, and ordering
- **Striking Hero Section** — Bold, compelling headline with accent design elements
- **Why Choose Us** — Three highlight cards explaining the brand promise
- **Modern Menu Layout** — Four categories with icons, item badges, and descriptions
- **Info Cards** — Location, hours, and delivery information with visual hierarchy
- **Integrated Map** — Embedded Google Map for easy navigation
- **Responsive Design** — Perfectly optimized for desktop, tablet, and mobile
- **Smooth Animations** — Subtle hover effects and transitions throughout
- **Professional Styling** — Modern color palette, typography, and spacing

## 🎨 Design Highlights

### Color Scheme
- **Primary Red** (#c0392b) — Bold, appetizing brand color
- **Gold Accent** (#e8a54d) — Premium, warm highlights
- **Light Background** (#f9f7f4) — Warm, inviting feel
- **Dark Text** (#1a1a1a) — Excellent readability

### Key Sections
1. **Navbar** — Sticky navigation with logo and quick CTA
2. **Hero** — Eye-catching headline, features, and dual CTAs
3. **Highlights** — Three benefit cards (Premium, Fresh, Authentic)
4. **Menu** — Four organized categories with emojis, badges, descriptions
5. **Location** — Info cards + embedded map + hours
6. **Final CTA** — Strong call-to-action banner
7. **Footer** — Brand info and links

### Interactive Elements
- Hover effects on cards (lift & shadow)
- Smooth button transitions with transforms
- Sticky navbar with backdrop blur
- Smooth scroll behavior
- Mobile-optimized layout

## 🔧 Customization

### Menu Items
Edit the menu categories in `index.html`:
- Search for `.menu-category` sections
- Update item names and descriptions
- Add/remove items as needed
- Badges available: "Most Popular", "Spicy"

### Colors
Edit CSS variables in `styles.css` (`:root` section):
```css
--primary-red: #c0392b
--accent-gold: #e8a54d
--text-dark: #1a1a1a
/* etc */
```

### Hours
Update hours in the "Hours" info card in `index.html`

### Contact
Update phone/address in footer and location section
